About this project

This project is supposed to demonstrate some of the following:

    • Responsive Layout
      1. Grid Layout
      2. Flex Layout
      3. Media Queries
      4. Uses relative units, whenever possible.
    • Animations/Transitions
  • JavaScript ES6
    • Fetch from API
      1. Simulate GET and POST from pretend database
    • Light/Dark Theme (persist between visits with local storage)
    • Event Listeners
  • HTML
    • Forms
    • FontAwesome Icons
