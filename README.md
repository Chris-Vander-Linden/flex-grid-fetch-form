This project was created to practice and demonstrate the following:

* CSS3 
  * DART Sass

    * Functions
    * Mixins
  * Responsive Layout

  * Animations/Transitions
* JavaScript ES6 
  * Fetch from API

  * Light/Dark Theme (persist between visits with local storage)
  * Event Listeners
* HTML 
  * Forms
  * FontAwesome Icons
* NPM 
  * Creating a Node Package
* Version Control 
  * Git/GitHub
